# Templates

Copy these to your project for agent context and report generation.

| Template | Purpose |
|----------|---------|
| [product-marketing-context.md](product-marketing-context.md) | Product, audience, brand — skills read automatically from `.cursor/` or `.claude/` |
| [skills-task-progress.md](skills-task-progress.md) | Task status (pending/done), priority — agent suggests next steps |
| [keyword-report.md](keyword-report.md) | Output format for keyword reports (used by [tools/](../tools/README.md)) |
| [competitor-report.md](competitor-report.md) | Output format for competitor reports (used by [tools/](../tools/README.md)) |
